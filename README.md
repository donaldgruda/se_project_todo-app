# Simple Todo App

A simple to-do list application that allows users to manage tasks with due dates.

## Functionality

Users can view a list of pre-loaded to-do items, add new ones via a popup form, mark items as completed or uncompleted, and delete items. Form validation ensures inputs meet requirements before submission.

## Technology

- HTML, CSS, JavaScript
- Object-Oriented Programming with ES6 classes
- BEM methodology for CSS
- UUID package for generating unique IDs
- Deployed via GitHub Pages
